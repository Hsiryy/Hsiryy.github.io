PHP变量规则

  • 变量名不能包含空格和特殊符号
  • 区分大小写
  • 必须以字母或者下划线字符开始
  • php语句和php变量都是区分大小写的

php没有声明变量的命令

变量在第一次赋值的时候被创建

PHP是一门弱类型语言

php会根据变量的值，自动把变量转换为正确的数据类型

PHP变量作用域

有四种不同的变量作用域

  • local
  • global
  • static
  • parameter

局部和全局作用域

局部不能引用全区变量的数据 相反亦然

<?php
$x=5;  //全局变量
// 函数框架，函数（大盒子（处理代码））
function myTest(){
    $y=10; //局部变量
    echo "<p>Text<p>";
    echo "变量 x 为：$x";
    echo "<br>";
    echo "变量 y 为 $y";
}

 // 调用函数 
 myTest();   

echo "<p>Text<p>";
echo "变量 x 为：$x";
echo "<br>";
echo "变量 y 为 $y";

?>

gobal

<?php
$x=5;  
$y=10; 

function myText(){
    global $x,$y,$z;
    $z=$x+$y;
    echo $z;
}

myText();
echo $z;
?>

Static

<?php
function myText(){
    static $x=0,$y=1;
    echo "第 $y 次：";

}
myText();
myText();
myText();
myText();
